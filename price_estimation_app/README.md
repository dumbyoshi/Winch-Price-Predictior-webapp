## Run application in local

## SETUP
    download and install python2 or python3
    $ pip install pandas
    $ pip install dash
    $ pip install dash_bootstrap_components
    $ pip install sklearn
    $ pip install xgboost

## Start application
    $ python web_app.py

    application runs in http://127.0.0.1:8030/
    
    To stop the application:
    ctrl + c
